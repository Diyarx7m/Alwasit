# Alwasit Turizm — Luxury Tourism Platform

A premium, RTL-first tourism website built for **Alwasit Turizm**, featuring dynamic category rendering, responsive mobile-first components, and seamless WhatsApp booking integration. Designed to showcase luxury travel experiences across Turkey with a focus on elegance, performance, and conversion.

## ✨ Features

- **Dynamic Category System**: 7 service categories (Tours, Medical Tourism, Hotels, Apartments, Cottages, Flights, Cars) rendered via a single, reusable component architecture.
- **Responsive Adaptive UI**: Desktop grid layout + mobile-optimized Swiper carousels with automatic breakpoint detection.
- **WhatsApp-First Booking**: Form submissions generate pre-formatted WhatsApp messages with structured booking data — zero backend required.
- **Performance-Optimized Animations**: 
  - Intersection Observer for scroll-triggered section reveals
  - Lightweight counter animations for key statistics
  - CSS hardware-accelerated transitions (no layout thrashing)
- **RTL-First Design**: Full Arabic language support with `Tajawal` font, logical spacing, and mirrored interactions.
- **Glassmorphism Header**: Subtle blur backdrop with scroll-aware styling for premium visual hierarchy.
- **Zero-Build Setup**: Pure HTML/CSS/JS — no bundler required. Drop on any static host and go.
